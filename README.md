# LeetCode to GitHub Browser Extension  

## Installation Instructions  
1. **Download the Extension**  
   - Go to the [Releases](https://github.com/uday-1602/LeetCode-to-GitHub/releases) page of this repository.  
   - Download the latest release of the browser extension.  
   
2. **Add the Extension to Your Browser**  
   - For Google Chrome:  
     - Open Chrome and navigate to `chrome://extensions/`.  
     - Enable "Developer mode" using the toggle switch in the upper right corner.  
     - Click on "Load unpacked" and select the folder containing the downloaded extension files.  
   
## Implementation Instructions  
1. **Configuration**  
   - After installation, click on the extension icon in the browser toolbar.  
   - Navigate to the settings page to configure your GitHub API token and other necessary settings.  
   
2. **Usage**  
   - Visit your LeetCode account and select the problem you want to solve.  
   - Once you've solved a problem, click the extension icon and choose "Upload to GitHub".  
   - Select the desired repository and branch to upload the code snippet.  
   - Confirm and check your GitHub repository to see the uploaded code.  

## Troubleshooting  
- If the extension is not working correctly, ensure that you have the necessary permissions set in your browser.  
- Check the console for any error messages and report them on the GitHub issue tracker.
